Snowball

March 2015 - September 2015

Description

Snowball is WordPress plugin that makes it possible for anyone to create immersive articles for the web. Snowball tries to hit the sweet spot between easy Graphical Editors and powerful Code Editors giving a user complete control over the customization of a webpage while maintining enough abstraction to not make the development process too overwhelming.

opacityChangeOnScroll.js

March 2015 -

Description

opacityChangeOnScroll is Javascript/jQuery library that makes it really easy for a user to bind the opacity(currently. Thinking about supporting other properties too) of an element on the page to the scroll event by just assigning HTML classes.

Language(s) Used

  • Javascript (jQuery)

Purpose

  • Give users the abilty to bind the scroll event to css properties by just assigning HTML classes
  • Ease of use
  • Complete control over the change the scroll bring about useing data-attributes
